## Description (max 4000 chars)
Tenjin is a personal study companion for vocabulary, phrases, and anything
worth remembering. Build a library of words enriched with formatted text, math
formulas, images, audio, and video. Group them into decks and review with a
proven spaced-repetition algorithm that schedules each card based on how well
you know it.

Everything stays on your device. No account required, no ads, no tracking.

Features:
- Rich content blocks: text, LaTeX formulas, images, audio, video
- Spaced-repetition reviews with adaptive scheduling
- Smart decks that auto-populate from tags
- Full-text search across words, tags, and decks
- Light and dark themes
- Multi-language interface
- Import and export your entire collection as a single file
- Review reminders

## Review notes
Tenjin stores all data locally in an on-device SQLite database. The optional
"Send feedback" feature posts a user-written bug report (and, if the user opts
in, diagnostic logs) to our endpoint. No account is required to use any feature.
Deck sync is shown as "Coming soon" and is not yet functional.
